Vamos a rellenar la base de datos con Seed. Para ello lo que hemos hecho ha sido mediante el composer instalar Faker, un proyecto que nos permite falsificar los datos que vamos a introducir en la base de datos. Mediante "composer require fzaninotto/faker" lo añadimos a nuestro proyecto de laravel. Para crear una clase de Seeder mediante referencias, se hará a traves de la consola de comandos mediante "php artisan make:seeder <nombre_de_la_clase>". Dentro de esta clase en la segunda linea escribimos "require_once 'vendor/fzaninotto/faker/src/autoload.php';
". luego creamos una variable que sea faker la cual hara referencia al proyecto que nos hemos bajado y asi poder utilizarlo. Creamos todo lo que necesitamos y una vez terminado lo que hacemos es mediante la consola de comandos ejecutarlo. Para ello usaremos "php artisan db:seed".  Si hemos modificado la base de datos o si hemos hecho un seeder pero ha dado error que se ha tenido que solucionar, lo recomendado es hacer "php artisan migrate:refresh --seed".
